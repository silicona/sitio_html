Etiquetas HTML importantes

La imagen tambien es un elemento inline

Esta página está diseñada como una referencia rápida de algunas de las más comunes etiquetas cubiertas en Learn Enough To Be Dangerous. En el proceso de hacerlo, aprenderemos como hacer tablas HTML mediante la etiqueta tabla y otras relacionadas.

Las tablas más abajo no incluyen todas las etiquetas HTML, pero hacen una lista de las más importantes.

Elementos Block

Etiqueta Nombre Proposito
h1–h6 Cabeceras Incluye una cabecera (niveles 1–6)
p Párrafo Incluye un parrafo de texto
table Tabla Incluye una tabla
tr Fila de tabla Incluye una fila de datos
th Cabecera de tabla Hace una cabecera de tabla
td Datos de tabla Incluye una celda de datos en la tabla
div Bloque de contenido Estructura el contenido como BLock

Inline Elements

Etiqueta Nombre Proposito Ejemplo Resultado
em emphasized Hace texto enfatizado <em>tsofisticación técnica</em> sofisticación técnica
strong strong Hace texto strong <strong>at least a billion people</strong> at least a billion people
a anchor Hace hyperlinks <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img image Incluye una imagen <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
span span Su contenido es un elemento Inline <span>contenido en linea con el texto</span> contenido en linea con el texto
